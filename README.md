# CampusThrift

CampusThrift is a beginner-friendly MERN marketplace starter for campus item buying and selling. It uses React for the frontend, Express and MongoDB for the backend, Tailwind for styling, Cloudinary for image uploads, and the Gemini API for AI-powered listing help.

## Folder structure

- client/ - React frontend app
- server/ - Express backend API
- package.json - root scripts for running both apps

## Quick start

1. Install dependencies: npm install
2. Start both apps: npm run dev
3. Visit the client at http://localhost:5173 and the API at http://localhost:5000
